# X-AI-Navigator

AIによって100%コードが生成された、X (旧Twitter) をキーボードで操作するためのユーザースクリプトです。

## 概要

このスクリプトは、X (旧Twitter) のブラウジングを、より快適に行うことを目指して作成しました。

このスクリプトを導入することにより、タイムラインの閲覧やいいね、リツイート等の主要なアクションをキーボードのみで完結させることが可能となります。

本プロジェクトの最も大きな特徴として、実装されたJavaScriptコードの全てが、人間による直接の記述ではなく、AI (Google Gemini) との対話を通して生成された点が挙げられます。

## 機能 (キーボードショートカット)

本スクリプトで利用可能なキーボードショートカットの一覧は、以下の通りです。

| キー                 | アクション                                              |
| -------------------- | ------------------------------------------------------- |
| `↑` `↓`             | ツイートを選択して上下に移動                            |
| `→`                 | いいね / いいね解除                                     |
| `←`                 | リツイート / リツイート解除                             |
| `b`                  | 選択中のツイートの投稿者をブロック                      |
| `<`                  | ポスト作成画面を開く                                    |
| `右 Alt`             | 表示中のポスト / 返信ボタンをクリック                   |
| `右 Shift`           | ブラウザの「戻る」を実行                                |
| `右 Ctrl`            | ツイートを新しいタブで開く (引用の場合は引用元へ移動) |
| `PageUp` / `PageDown`| タイムラインのタブを切り替え (おすすめ/フォロー中など) |

## 制作背景

普段からX (旧Twitter) を頻繁に利用する中で、マウスとキーボードを行き来する操作に煩わしさを感じており、キーボードだけで完結する快適な操作環境を作りたい、と考えたのが開発のきっかけです。

それに加え、今注目されているAIによるコード生成技術が、どの程度実用的なアプリケーション開発に寄与できるのか、という実験も目的としました。

実際の開発では、私は一切コードを記述せず、必要な機能の仕様や、目的などを情報をプロンプトとしてAIに指示し、その生成結果を検証・修正依頼するというプロセスを繰り返しました。この経験を通して、AIを活用した新しい開発手法の可能性を学ぶことができました。

## 使用技術

- **言語**: JavaScript
- **主要API**: DOM Manipulation, `MutationObserver`
- **コード生成AI**: Google Gemini (本スクリプトの全コードを生成)
- **実行環境**: ユーザースクリプトマネージャ (Tampermonkey等)
